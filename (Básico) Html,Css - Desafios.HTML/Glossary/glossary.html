Estrutura HTML

!DOCTYPE html
Usado para descrever que o conteúdo utilizado na página, que está sendo usado o HTML5.
metadado
Metadado é um dado que tem outros dados dentro, significa que dentro desse dado podem ter links, imagens, videos, áudios, botões, etc..
head
Tag Head é um elemento metadado, fica entre o !DOCTYPE e o body, nele é anexado informações como título da página. Ex: Google. Além do ícone da página, o estilo de fonte usada, a qual linguágem vai ser feita a lógica que será usada, etc...
body
Conhecido como o corpo da página, onde é posto os conteúdos principal vistos pelo cliente.
header
Tag cabeçalho, onde é posto a parte superior da página, normalmente como o ícone, parte de navegação, configuração, login...
nav
Nav é uma tag semântica para a navegação da página, tem pessoas que dizem que o Nav pode ser usado dentro de um header, outros dizem que não, também pode ser usada no footer.
footer
Tag footer, é o rodapé geralmente contem o mapa do site, parcerias, informações extras, termos de uso, informações de contato.
